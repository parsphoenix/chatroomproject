# Chat PHP Project (chatphpproject)

یک سیستم چت و تماس تصویری پیشرفته با استفاده از PHP و WebRTC.

## امکانات پروژه:
- **تماس تصویری و صوتی**: با استفاده از تکنولوژی WebRTC.
- **تماس گروهی**: پشتیبانی از تماس‌های گروهی (Mesh Architecture).
- **مدیریت پهنای باند**: محدودیت بیت‌ریت خودکار برای کیفیت‌های مختلف (240p, 360p, 480p).
- **پنل مدیریت**: مدیریت کاربران، پیام‌ها و وضعیت سیستم.
- **تم‌های روشن و تاریک**: قابلیت انتخاب تم توسط کاربر و ذخیره در دیتابیس.
- **سیستم آپلود پیشرفته**: آپلود تصویر پروفایل و فایل در چت با محدودیت حجم.
- **طراحی ریسپانسیو**: نمایش بهینه در تمامی پلتفرم‌ها (موبایل، تبلت، دسکتاپ).
- **سیستم چندزبانه**: پشتیبانی از زبان‌های فارسی و انگلیسی.

## نحوه نصب:
1. محتویات پروژه را در سرور خود (مانند XAMPP) قرار دهید.
2. دیتابیس را ایجاد کرده و اطلاعات آن را در فایل `config/db.php` (که پس از نصب ایجاد می‌شود) قرار دهید.
3. فایل `install.php` را در مرورگر اجرا کنید.
